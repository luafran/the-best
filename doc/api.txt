Get best of something
=====================

1 User start typing a question

2 While typing system suggest questions (auto complete questions)
	API REQUEST: GET /api/suggestions?type=q&text=be
	API RESPONSE: 200 {"suggestions": [{"text": "beer"}]}

3 User submit question
	API REQUEST: GET /api/best_answer?q=beer

4 System search for best answer for the given question and is found
    API RESPONSE: {"items": [{"q": "beer", "a": "Imperial", "id": "1"}]}

5 System display "Answer to the question"

Alternatives
------------

2a ALT: There is no suggestions for the given text
    API RESPONSE: 200 {"suggestions": []}


4a	System search for best answer for the given question and no item is found (first time question)
    API RESPONSE: {"items": []}

4a1 New question is added to the system
    API REQUEST: POST /api/items {"item":{"q": "beer", "a": null}
    API RESPONSE: {}

4a2 System display "New question"


4b System search for best answer for the given question and item is found with no answer
    API RESPONSE: {"items": [{"q": "beer", "a": null}]}

4b1 System display "No answer yet"


Answer system question
======================

1 System search a question for the user to answer
    API REQUEST: GET /api/user_question

2 System select an existing question with no answer yet
    API RESPONSE: {"items": [{"id": "1", "q": "beer", "a": null}]}

3 User start typing an answer

4 While typing, system suggest answers (auto complete answers)
    API REQUEST: GET /api/suggestions?type=a&q=beer&text=imp
	API RESPONSE: 200 {"suggestions": [{"text": "imperial"}]}

5 User submit answer to a question with no answer yet
    API REQUEST: PUT /api/items/1 {"item": {"id": "1", "q": "beer", "a": "Imperial"}}
    API RESPONSE: {}


Alternatives
------------

2a System select an existing question that has answer(s)
    API RESPONSE: {"items": [{"q": "beer"}]}


3a User select to skip system question

3a1 End of use case


4a There is no suggestions for the given text
    API RESPONSE: 200 {"suggestions": []}


5a User submit new answer to a question that has answer(s)
    API REQUEST: POST /api/user_answer {"item": {"q": "beer", "a": "Heineken"}}
    API RESPONSE: {}

5a1 System create a new item {"q": "beer", "a": "Heineken"}}


5b User submit existing answer to a question that has answer(s)
    API REQUEST: POST /api/user_answer {"item": {"q": "beer", "a": "Imperial"}}
    API RESPONSE: {}

5b1 System add a vote to item with "q": "beer" and "a": "Imperial"


Old Stuff
=========

Get suggestions to autocomplete a question

GET /api/suggestions/question?text=ho
{"suggestions": [{"text": "hotel"}, {"text": "hospital"}]}


Get suggestions to autocomplete an answer

GET /api/suggestions/answer?q=hotel&text=ho
{"suggestions": [{"text": "Holiday Inn"}]}


Get a question for the user to answer

GET /api/user_question
{"item": {"q": "Beach"}}

Post user answer for a given question

POST /api/user_answer
{"item": {"q": "Beach", "a": "Mariscal"}}


Get "the best" (tm) answer to a given question

GET /api/best_answer?q=hotel
{"item": {"q": "hotel", "a": "Holiday Inn"}}

GET /api/best_answer?q=beer
{"item": {"q": "beer", "a": "Imperial"}}
